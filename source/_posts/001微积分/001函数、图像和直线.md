微积分是研究变化的数学，而**函数**是描述变化关系的基石，也就是说，函数是微积分的研究对象。

核心：本章是微积分的基础，重点在于理解“函数是变量间的对应关系”，以及不同函数的表达、图像和特性，为后续导数、积分的学习打基础。

# 函数

**函数是关系的量化描述**

## 1. 定义

对于两个集合A和B，如果存在一种对应关系f，使得对于集合A中的**每一个**元素x，在集合B中都有**唯一确定**的元素y与之对应，那么f就是从A到B的一个函数。

**核心概念**

- **定义域**：函数能接受的所有输入x的集合（“x的合法取值范围”），比如f(x)=1/x中，x不能为0，定义域就是x≠0。
- **值域**：函数所有可能输出y的集合（“y的实际结果范围”），比如f(x)=x²中，y永远是非负数，值域就是y≥0。
- **自变量与因变量**：x是自变量（主动变化的量），y是因变量（随x变化而变化的量）。

**表示方法**

- **解析法**：用数学公式表示，如 `f(x) = x² + 1`
- **图像法**：在坐标系中用曲线表示
- **枚举法**：用表格列出x和y的对应值

**微积分中的重要性**

- 微积分的核心研究对象就是函数
- 导数研究函数的变化率（瞬时速度）
- 积分研究函数的累积效应（曲线下面积）

**区间表示法**

用来简洁表示定义域、值域的范围，核心是“括号表边界是否包含”：

| 区间类型             | 符号表示           | 含义（x的范围）        | 例子                                |
| :------------------- | :----------------- | :--------------------- | :---------------------------------- |
| 闭区间（包含边界）   | [a, b]             | a ≤ x ≤ b              | [1,3] 即x取1、2、3及中间所有数      |
| 开区间（不包含边界） | (a, b)             | a < x < b              | (1,3) 即x取1和3之间的数，不包括1和3 |
| 半开半闭区间         | [a, b) 或 (a, b]   | a ≤ x < b 或 a < x ≤ b | [1,3) 包含1，不包含3                |
| 无穷区间             | (-∞, a] 或 (b, +∞) | x ≤ a 或 x > b         | (-∞, 2] 即x是所有小于等于2的数      |

注意：无穷大（∞）永远用开括号，因为它不是一个具体的数，无法“包含”。

**可视化示例**

## 2. 如何求定义域

定义域是“x的合法取值”，核心是排除让函数“无意义”的x，常见规则：

- **分式函数**：分母不能为0 → 例：f(x)=1/(x-2)，定义域是x≠2（即(-∞,2)∪(2,+∞)）。
- **偶次根号函数**（如√x）：根号内的数≥0 → 例：f(x)=√(3x-6)，需3x-6≥0 → x≥2，定义域[2,+∞)。
- **对数函数**（如lnx）：真数必须>0 → 例：f(x)=ln(x+1)，定义域x+1>0 → x>-1（(-1,+∞)）。
- **实际问题**：结合场景，比如“人数”不能为负数，“时间”不能为负 → 例：f(t)=2t表示t小时后的路程，定义域t≥0。

技巧：先默认x可取所有实数，再排除“违规”的x，剩下的就是定义域。

## 3. 如何求值域

值域是“y的所有可能结果”，常用方法有3种：

- **代数法**：通过变形函数表达式，找出y的范围。        例1：f(x)=2x+1，x∈[1,3] → 当x=1时y=3，x=3时y=7，值域[3,7]（一次函数单调，直接取端点值）。例2：f(x)=x²-2x+3 → 配方为f(x)=(x-1)²+2 → (x-1)²≥0，所以y≥2，值域[2,+∞)。
- **换元法**：把复杂部分换成新变量，简化计算。        例：f(x)=√x + 1 → 设t=√x（t≥0），则y=t+1 → t≥0 → y≥1，值域[1,+∞)。
- **图像法**：通过函数图像，看y轴上的“覆盖范围”

函数图像是“所有(x,f(x))点的集合”，值域就是图像在*y轴上的投影范围*——即图像能“碰到”的所有y值。

步骤：1. 画出函数图像；2. 观察图像在y轴方向的最高点和最低点；3. 确定y的范围（包含端点则用闭区间，否则用开区间）。

例子：

- f(x)=|x|（绝对值函数）：图像是“V”型，顶点在(0,0)，向上无限延伸 → 值域[0,+∞)。
- f(x)=x²-4x+5：图像是开口向上的抛物线，顶点在(2,1) → 值域[1,+∞)。
- f(x)=sinx：图像在y=-1到y=1之间来回波动 → 值域[-1,1]。

图像参考：可搜索“绝对值函数图像”“正弦函数图像”，核心记清图像的“高低边界”。

## 4. 垂线检验

用来判断“一个图像是不是函数的图像”——因为函数要求“一个x对应唯一y”，所以：

**规则**：在图像上画任意一条垂直于x轴的直线，如果这条直线*只和图像有一个交点*，就是函数图像；如果有两个或更多交点，就不是。

例子：圆x²+y²=1 → 画x=0的垂线（y轴），和圆交于(0,1)和(0,-1)两个点 → 不是函数图像；而抛物线y=x²，任意垂线都只交一个点 → 是函数图像。

## 5. 特点

- 唯一性：一个x只能对应一个y（核心特点，由垂线检验保证）。
- 确定性：对于给定的x，y的结果是确定的，不会“有时是a，有时是b”。
- 方向性：是x→y的对应，不是y→x（y→x的对应是反函数的事）。

## 6.性质

- **单调性**：
  - 单调递增：x₁ < x₂ ⇒ f(x₁) ≤ f(x₂)
  - 严格单调递增：x₁ < x₂ ⇒ f(x₁) < f(x₂)
  - 单调递减：x₁ < x₂ ⇒ f(x₁) ≥ f(x₂)
  - 严格单调递减：x₁ < x₂ ⇒ f(x₁) > f(x₂)
  - *微积分关联*：导数正负决定单调性
- **奇偶性**：
  - 奇函数：f(-x) = -f(x)，图像关于原点对称
  - 偶函数：f(-x) = f(x)，图像关于y轴对称
  - *微积分关联*：简化计算，对称区间上奇函数的积分为0
- **周期性**：存在正数T，使得f(x+T) = f(x)
  - *微积分关联*：周期函数的导数、积分也具有周期性
- **有界性**：函数值是否在某个范围内

备注：具有单调递增的函数叫单调递增函数等延伸的函数定义。

```python
    ax.grid(True, alpha=0.3)
    ax.axhline(y=0, color='k', linestyle='-', alpha=0.3)

plt.show()
```



# 反函数

## 1. 定义

反函数是“逆转函数的对应关系”——如果函数f把x对应到y（y=f(x)），那么反函数f⁻¹就把y对应回x（x=f⁻¹(y)）。

注意：f⁻¹读作“f的逆”，不是“f的负一次方”，和1/f(x)完全不同。

例：f(x)=2x+1（x→2x+1），反函数f⁻¹(y)=(y-1)/2（y→(y-1)/2）；当x=3时f(3)=7，那么f⁻¹(7)=3。

## 2. 水平线检验

不是所有函数都有反函数——只有“一个y也只对应一个x”的函数，才有反函数（这种函数叫“一一对应函数”）。水平线检验就是判断函数是否一一对应：

**规则**：在函数图像上画任意一条水平于x轴的直线，如果这条直线*只和图像有一个交点*，函数就有反函数；如果有两个或更多交点，就没有。

例子：f(x)=x² → 画y=4的水平线，和图像交于(2,4)和(-2,4) → 没有反函数；但如果限定f(x)=x²（x≥0，只取右半部分抛物线），水平线y=4只交(2,4) → 有反函数。

## 3. 如何求反函数

核心步骤：“交换x和y，再解出y”，共3步：

1. 把y=f(x)中的x和y交换位置，得到x=f(y)；
2. 从x=f(y)中解出y，得到y=f⁻¹(x)；
3. 确定反函数的定义域（即原函数的值域）。

例：求f(x)=3x-2（x∈R）的反函数：

1. 交换x和y：x=3y-2；
2. 解y：3y=x+2 → y=(x+2)/3；
3. 原函数值域是R，所以反函数定义域是R → 反函数f⁻¹(x)=(x+2)/3。

## 4. 特点

- 反函数的定义域 = 原函数的值域；反函数的值域 = 原函数的定义域（“交换”的必然结果）。
- f和f⁻¹的图像关于直线y=x对称 → 例：f(x)=2x+1的图像和f⁻¹(x)=(x-1)/2的图像，画出来会关于y=x翻折。
- 复合性质：f(f⁻¹(x))=x，f⁻¹(f(x))=x → 把反函数代入原函数，结果就是x本身（“逆转再逆转，回到原点”）。

# 复合函数

## 1. 定义

复合函数是“把一个函数的输出作为另一个函数的输入”，就像“函数套函数”。记为(f∘g)(x)，读作“f复合g关于x”，意思是先算g(x)，再把结果代入f(x)，即(f∘g)(x)=f(g(x))。

例：f(x)=x²，g(x)=2x+3 → 复合函数(f∘g)(x)=f(g(x))=f(2x+3)=(2x+3)²=4x²+12x+9；而(g∘f)(x)=g(f(x))=g(x²)=2x²+3，注意(f∘g)和(g∘f)通常不一样。

定义域要求：g(x)的结果必须在f(x)的定义域内 → 例：f(x)=√x（定义域x≥0），g(x)=x-1 → (f∘g)(x)=√(x-1)，需x-1≥0 → 定义域x≥1。

# 奇偶函数

## 1. 定义

判断函数图像是否关于原点或y轴对称的依据，前提是函数定义域关于原点对称（即x在定义域内时，-x也在定义域内，否则无法判断）：

- **偶函数**：对定义域内所有x，都有f(-x)=f(x) → 图像关于y轴对称。
- **奇函数**：对定义域内所有x，都有f(-x)=-f(x) → 图像关于原点对称。

例：f(x)=x² → f(-x)=(-x)²=x²=f(x) → 偶函数；f(x)=x³ → f(-x)=(-x)³=-x³=-f(x) → 奇函数；f(x)=2x+1 → f(-x)=-2x+1≠f(x)且≠-f(x) → 非奇非偶。

## 2. 性质

- 偶函数的图像：y轴是对称轴，比如x²、|x|的图像，左边和右边完全一样。
- 奇函数的图像：原点是对称中心，把图像绕原点旋转180°后和原图像重合；且若奇函数在x=0处有定义，一定有f(0)=0（因为f(-0)=-f(0) → f(0)=-f(0) → 2f(0)=0）。
- 运算性质：偶+偶=偶，奇+奇=奇，偶×偶=偶，奇×奇=偶，偶×奇=奇。

# 线性函数的图像

线性函数是“最高次项为一次”的函数，核心形式：y = mx + b（m和b是常数），其中：

- **m**：斜率，决定直线的“倾斜方向和程度”——        m>0：直线从左下往右上倾斜（x增大，y增大）；m越大，倾斜越陡；m<0：直线从左上往右下倾斜（x增大，y减小）；m越小（负得越多），倾斜越陡；m=0：直线水平（y=b，不随x变化）；m不存在：直线垂直于x轴（x=a，不是函数，因为垂线检验不通过）。
- **b**：y轴截距，是直线和y轴交点的纵坐标——交点坐标是(0, b)，比如y=2x+3，和y轴交于(0,3)；y=3x-1，和y轴交于(0,-1)。

图像特点：所有线性函数的图像都是“直线”，画直线只需找两个点就能确定：

例：画y=2x+1的图像——找x=0时y=1（点(0,1)），x=1时y=3（点(1,3)），连接两点就是直线。

特殊情况：当b=0时，y=mx，直线过原点(0,0)，是“正比例函数”（线性函数的特例）。

# 常见函数及其图像

## 1. 多项式函数

形式：f(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀（aₙ≠0，n是非负整数，叫多项式的“次数”），特点是定义域都是R（全体实数）。

- **一次多项式**：n=1，即线性函数y=mx+b，图像是直线（前面已讲）。
- **二次多项式**：n=2，即二次函数y=ax²+bx+c（a≠0），图像是“抛物线”——        a>0：开口向上，有最低点（顶点）；a<0：开口向下，有最高点；顶点坐标：(-b/(2a), f(-b/(2a)))，是图像的对称轴（x=-b/(2a)）。
- **三次多项式**：n=3，如y=x³，图像过原点，在R上单调递增，定义域和值域都是R。

图像参考：搜索“二次函数抛物线图像”“三次函数x³图像”，重点记开口方向和顶点。

## 2. 有理函数

形式：f(x) = P(x)/Q(x)，其中P(x)和Q(x)是多项式，且Q(x)≠0（分母不能为0），核心是“分式形式”。

常见例子：

- **反比例函数**：f(x)=k/x（k≠0），定义域x≠0，图像是“双曲线”——        k>0：双曲线在一、三象限；k<0：在二、四象限；特点：图像无限靠近x轴和y轴，但永远不相交（x轴和y轴是“渐近线”）。
- **简单分式**：f(x)=(x+1)/(x-2)，定义域x≠2，渐近线是x=2（垂直渐近线）和y=1（水平渐近线）。

## 3. 指数函数和对数函数

两者是“互反函数”，核心底数a>0且a≠1（保证定义域是R或正实数）。

| 类型     | 形式                  | 定义域             | 值域               | 图像特点                                                  |
| :------- | :-------------------- | :----------------- | :----------------- | :-------------------------------------------------------- |
| 指数函数 | f(x)=aˣ               | R（全体实数）      | (0, +∞)（y永远正） | 过点(0,1)；a>1时递增（如2ˣ），0<a<1时递减（如(1/2)ˣ）     |
| 对数函数 | f(x)=logₐx（a是底数） | (0, +∞)（x必须正） | R（全体实数）      | 过点(1,0)；a>1时递增（如lnx，底数是e≈2.718），0<a<1时递减 |

关键关系：logₐx = y 等价于 aʸ = x（对数是指数的逆运算），所以它们的图像关于y=x对称。

## 4. 三角函数

核心是描述“角度与边长的关系”，常见的有正弦（sinx）、余弦（cosx）、正切（tanx），定义域和图像各有特点：

- **sinx（正弦函数）**：        定义域R，值域[-1,1]；图像：“波浪线”，周期2π（每隔2π重复一次）；过原点(0,0)，关于原点对称（奇函数）；关键点：(0,0)、(π/2,1)、(π,0)、(3π/2,-1)、(2π,0)。
- **cosx（余弦函数）**：        定义域R，值域[-1,1]；图像：“波浪线”，周期2π；过点(0,1)，关于y轴对称（偶函数）；关键点：(0,1)、(π/2,0)、(π,-1)、(3π/2,0)、(2π,1)。
- **tanx（正切函数）**：        形式tanx=sinx/cosx，定义域x≠π/2 +kπ（k是整数，因为cosx不能为0）；值域R，周期π；图像是“间断的直线”，在x=π/2 +kπ处有“垂直渐近线”。

图像参考：搜索“正弦函数图像”“余弦函数图像”，重点记周期、值域和关键点。

## 5. 绝对值函数

形式：f(x)=|x|，核心是“把负数变正，正数和0不变”，即：

f(x) = x（当x≥0时）；f(x) = -x（当x<0时）

特点：

- 定义域R，值域[0,+∞)；
- 图像：“V”型，顶点在(0,0)，关于y轴对称（偶函数）；
- 变形：f(x)=|x-a|+b，图像是“V”型顶点平移到(a,b)，比如|x-2|+1的顶点在(2,1)。

# 基本初等函数家族

1. **幂函数**：y = xᵃ（a为实数）
2. **指数函数**：y = aˣ（a>0, a≠1）
3. **对数函数**：y = logₐx（a>0, a≠1）
4. **三角函数**：sin(x), cos(x), tan(x)等
5. **常数函数**：y = C

## 图像示例

```python
import numpy as np
import matplotlib.pyplot as plt

fig, axes = plt.subplots(2, 3, figsize=(16, 10))
x = np.linspace(0.1, 3, 100)  # 从0.1开始避免对数函数问题

# 1. 幂函数家族
powers = [0.5, 1, 2, 3]
colors = ['red', 'blue', 'green', 'purple']
for power, color in zip(powers, colors):
    y_power = x**power
    axes[0, 0].plot(x, y_power, color=color, linewidth=2, label=f'x^{power}')
axes[0, 0].set_title('幂函数家族')
axes[0, 0].legend()

# 2. 指数函数
bases = [0.5, 2, np.e]
colors = ['red', 'blue', 'green']
for base, color in zip(bases, colors):
    y_exp = base**x
    axes[0, 1].plot(x, y_exp, color=color, linewidth=2, label=f'{base:.2f}ˣ')
axes[0, 1].set_title('指数函数家族')
axes[0, 1].legend()

# 3. 对数函数
for base, color in zip(bases, colors):
    if base != 1:
        y_log = np.log(x) / np.log(base)  # 换底公式
        axes[0, 2].plot(x, y_log, color=color, linewidth=2, label=f'log_{base:.1f}(x)')
axes[0, 2].set_title('对数函数家族')
axes[0, 2].legend()

# 4. 三角函数
x_trig = np.linspace(-2*np.pi, 2*np.pi, 200)
axes[1, 0].plot(x_trig, np.sin(x_trig), 'r-', linewidth=2, label='sin(x)')
axes[1, 0].plot(x_trig, np.cos(x_trig), 'b-', linewidth=2, label='cos(x)')
axes[1, 0].set_title('三角函数')
axes[1, 0].legend()

# 5. 常数函数和线性函数
x_linear = np.linspace(-2, 2, 50)
axes[1, 1].plot(x_linear, np.ones_like(x_linear)*2, 'purple', linewidth=2, label='y = 2')
axes[1, 1].plot(x_linear, 0.5*x_linear + 1, 'orange', linewidth=2, label='y = 0.5x + 1')
axes[1, 1].set_title('常数函数和线性函数')
axes[1, 1].legend()

# 6. 函数组合示例
axes[1, 2].plot(x, np.exp(-x)*np.sin(5*x), 'teal', linewidth=2)
axes[1, 2].set_title('复合函数: e⁻ˣ·sin(5x)')

# 美化所有子图
for ax in axes.flat:
    ax.grid(True, alpha=0.3)
    ax.axhline(y=0, color='k', linestyle='-', alpha=0.3)
    ax.axvline(x=0, color='k', linestyle='-', alpha=0.3)

plt.tight_layout()
plt.show()
```

## 运算

- **四则运算**：加减乘除产生新函数
- **复合函数**：f(g(x))，一个函数的输出作为另一个函数的输入
- **函数变换**：
  - 平移：f(x ± a) 左右平移，f(x) ± a 上下平移
  - 伸缩：f(ax) 水平伸缩，a·f(x) 垂直伸缩
  - 反射：f(-x) 关于y轴反射，-f(x) 关于x轴反射

```python
import numpy as np
import matplotlib.pyplot as plt

# 演示函数变换
x = np.linspace(-3, 5, 200)
base_function = lambda x: x**2  # 基础函数 f(x) = x²

fig, axes = plt.subplots(2, 2, figsize=(14, 10))

# 原始函数
axes[0, 0].plot(x, base_function(x), 'k-', linewidth=3, label='f(x) = x²')
axes[0, 0].set_title('原始函数')
axes[0, 0].legend()

# 平移变换
axes[0, 1].plot(x, base_function(x), 'k-', linewidth=2, label='f(x)', alpha=0.5)
axes[0, 1].plot(x, base_function(x-2), 'r-', linewidth=2, label='f(x-2) 右移2单位')
axes[0, 1].plot(x, base_function(x)+1, 'b-', linewidth=2, label='f(x)+1 上移1单位')
axes[0, 1].set_title('平移变换')
axes[0, 1].legend()

# 伸缩变换
axes[1, 0].plot(x, base_function(x), 'k-', linewidth=2, label='f(x)', alpha=0.5)
axes[1, 0].plot(x, 2*base_function(x), 'r-', linewidth=2, label='2f(x) 垂直拉伸')
axes[1, 0].plot(x, base_function(0.5*x), 'b-', linewidth=2, label='f(0.5x) 水平拉伸')
axes[1, 0].set_title('伸缩变换')
axes[1, 0].legend()

# 反射变换
axes[1, 1].plot(x, base_function(x), 'k-', linewidth=2, label='f(x)', alpha=0.5)
axes[1, 1].plot(x, base_function(-x), 'r-', linewidth=2, label='f(-x) 关于y轴反射')
axes[1, 1].plot(x, -base_function(x), 'b-', linewidth=2, label='-f(x) 关于x轴反射')
axes[1, 1].set_title('反射变换')
axes[1, 1].legend()

for ax in axes.flat:
    ax.grid(True, alpha=0.3)
    ax.axhline(y=0, color='k', linestyle='-', alpha=0.3)
    ax.axvline(x=0, color='k', linestyle='-', alpha=0.3)
    ax.set_xlim(-3, 5)

plt.tight_layout()
plt.show()
```



# 七、总结

本章核心是“函数是x到y的唯一对应”，所有知识点都围绕这个核心展开：

1. 函数的定义和检验（垂线检验）是基础，定义域和值域是函数的“边界”；
2. 反函数是“逆对应”，需要水平线检验保证一一对应；
3. 复合函数是“函数套函数”，重点是输入输出的传递；
4. 奇偶函数通过f(-x)的关系判断，对应图像的对称性；
5. 线性函数是最简单的多项式，图像是直线，由斜率和截距决定；
6. 常见函数的图像和特性是后续学习的“素材”，需记住核心形状和范围。

关键技巧：学习函数时，“表达式→图像→性质”三者结合，看到表达式能想图像，看到图像能说性质，才能真正理解。
